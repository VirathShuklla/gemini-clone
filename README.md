# 🌌 Gemini Clone (React + Vite)

A sleek **Gemini-style AI chat** built with **React + Vite**.  
Fast dev server, tiny bundles, and a clean component architecture — ready to plug into the Gemini API (or any LLM backend).

---

## 🚀 Features

- ⚡ **Vite** dev server & HMR
- 💬 **Chat UI** with multi-turn context
- 🎨 **Modern UI** (CSS or Tailwind)
- 🧩 **Modular components**
- 🔐 **.env-based API key** handling

---

## 🧠 Tech Stack

- **Frontend:** React 18, Vite
- **Styling:** CSS / Tailwind (optional)
- **API:** Google Generative Language (Gemini) or custom backend
- **Deploy:** Vercel / Netlify / GitHub Pages / Azure Static Web Apps
